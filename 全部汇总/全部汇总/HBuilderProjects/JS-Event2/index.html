1.1绑定事件: a.添加事件属性 on+事件名 b.添加事件监听 标准浏览器下:addEventListener('事件名', 事件处理函数, 是否捕获); IE8-下: attachEvent('on事件名', 事件处理函数); 1.2取消事件绑定: a. on+事件名方式 = null; b. addEventListener(): removeEventListener(); attachEvent(): detachEvent(); 2.event事件对象 非火狐下直接用event,火狐下加一个形参 type事件类型 clientX/clientY: 鼠标基于页面左上角的位置 screenX/screenY: 鼠标基于屏幕左上角的位置 获取目标源: target/srcElement(IE下) keyCode: 键码值 key 案按键内容 altKey ctrlKey shiftKey 3.事件类型: 鼠标事件: click dbclick contextmenu 右击 mousedown mouseup mousemove mouseover mouseout mouseenter mouseleave 键盘事件: keydown keyup keypress 表单事件: focus blur change(变化且失焦) input(仅变化) select submit 其他事件: load error resize scroll 4.事件流: 捕获阶段--目标阶段(出阿发执行)--冒泡阶段 如何阻止事件冒泡: 标准下: event.stopPropagation(); IE下: event.cancelBubble = true; 5.取消默认行为: 标准下: event.preventDefault(); IE下: return false; 或者 event.returnValue = false; 用于a链接的href,右击,表单的默认行为
